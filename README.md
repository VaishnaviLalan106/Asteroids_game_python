# 🚀 Asteroids Game (Python)

A simple 2D Asteroids-style game built using Python and Pygame Zero.  
This project focuses on game logic, movement physics, and collision detection.
I made this project for my fun learning and to improve my skills.

---

## 🎮 Features

- 🚀 Rotating spaceship movement
- 💫 Smooth acceleration with friction
- 🔫 Shooting bullets with cooldown
- 🪨 Asteroids that split into smaller pieces
- 🌌 Infinite wrap-around screen
- 💥 Collision detection between objects

---

## ▶️ How to Run

```bash
pgzrun main.py
```

---

## 🎮 Controls

- ⬅️ / ➡️ → Rotate spaceship
- ⬆️ → Move forward
- ⬇️ → Move backward
- S → Shoot

---
## 🧠 Game Logic
- The spaceship moves based on its direction (angle-based movement)
- Bullets disappear after a short time
- Asteroids break into smaller pieces when hit
- Game resets if the spaceship collides with an asteroid
